# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-07-06

### Added

- Pomodoro timer with customizable focus/break durations
- Timestamp-based timer engine (accurate through sleep/tab switches)
- Task management (CRUD, priorities, labels, due dates)
- Statistics dashboard (daily, weekly, monthly views)
- Settings panel (theme, accent color, sound, notifications)
- Keyboard shortcuts (Space, S, R)
- PWA support with service worker
- Dark mode (system/light/dark)
- Responsive design (320px to 4K)
- Accessibility (WCAG AA, ARIA labels, keyboard nav)
- Sound effects via Web Audio API
- Browser notifications
- Mobile vibration support
- Animated progress ring
- Zustand state management with persistence
- REST API with Zod validation
- Prisma ORM with PostgreSQL
- Testing (Vitest, React Testing Library, Playwright)
- CI/CD with GitHub Actions
- Full documentation
